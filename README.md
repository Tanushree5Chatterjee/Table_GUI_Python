# Multiplication Table Generator  

## Technologies Used  
- Python  
- Tkinter  

## Project Description  
A simple GUI-based Python application developed using Tkinter to generate multiplication tables. The user enters a number, and the program displays its table from 1 to 10 in a structured format. It also provides options to reset input and exit the application.  

## Features  
- Generate multiplication table (1 to 10)  
- User-friendly GUI interface  
- Reset input and output  
- Exit application using Quit button  
- Basic input handling
